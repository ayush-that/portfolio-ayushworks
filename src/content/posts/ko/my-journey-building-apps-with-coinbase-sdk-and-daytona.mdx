**023: Take Daytona for a spin** 챌린지에 참여하기로 결정했고, 이 블로그에서 Daytona 경험을 공유합니다.

## Daytona란?

[Daytona](https://www.daytona.io/)는 셀프 호스팅되고 안전한 **오픈 소스** 개발 환경 관리자입니다.

![Daytona GitHub](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/6x6q69ztvir2vwo7yyux.png)

라이브러리 다운로드, 빌드, 업데이트, 컴파일, 테스트와 같은 추가 단계를 개발 프로세스에서 제거하고 **단일 명령**으로 코딩을 시작할 수 있게 해주는 간단하고 번거롭지 않은 **CLI 도구**입니다.

## Daytona 배우기

Daytona에 대한 기본 아이디어를 얻기 위해 먼저 [GitHub](https://github.com/daytonaio/daytona) 리포지토리를 열고 `README.md`를 읽었습니다. 활발한 개발과 13.5k 스타에 감명받았습니다. 그 후 [문서](https://www.daytona.io/docs/)와 [블로그](https://www.daytona.io/dotfiles)를 확인했습니다.

그런 다음 Daytona 테마에서 제공하는 것 중에서 퀘스트용 테마를 선택했습니다.

## 앱 빌드

퀘스트를 위해 Coinbase Wallet SDK를 사용하여 React 앱에서 Coinbase Wallet에 연결하기 위한 템플릿을 빌드했습니다.

![Coinbase Connect](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/shm1uaimxgzjj6dwwejo.png)

React + Vite 프로젝트를 설정하고 TypeScript를 활용하며 Coinbase SDK 문서를 빠르게 읽었습니다. 하룻밤 만에 MVP가 준비되었습니다. shadcn/ui와 TailwindCSS로 스타일링하고 Vercel에 배포했습니다.

Web3 프로젝트의 Coinbase 연결에 초점을 맞추고 있지만, 이것은 shadcn/ui 보일러플레이트로도 사용할 수 있습니다 ㅋㅋ.

## Daytona로 React와 Coinbase SDK 사용하기

이것이 블로그에서 가장 포괄적인 섹션이어야 하지만, Daytona의 놀랍도록 간단한 특성 덕분에 할 일이 많지 않습니다.

먼저 Daytona 설치는 쉽고 Windows에서도 마찬가지입니다. 단일 명령 설치로 완료됩니다.

```powershell
$architecture = if ($env:PROCESSOR_ARCHITECTURE -eq "AMD64") { "amd64" } else { "arm64" }
md -Force "$Env:APPDATA\bin\daytona"; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.SecurityProtocolType]'Tls,Tls11,Tls12';
Invoke-WebRequest -URI "https://download.daytona.io/daytona/v0.50/daytona-windows-$architecture.exe" -OutFile "$Env:APPDATA\bin\daytona\daytona.exe";
$env:Path += ";" + $Env:APPDATA + "\bin\daytona"; [Environment]::SetEnvironmentVariable("Path", $env:Path, [System.EnvironmentVariableTarget]::User);
daytona serve;
```

Linux 또는 MacOS를 사용하는 경우 이 명령으로 Daytona를 설치할 수 있습니다:

```bash
(curl -sf -L https://download.daytona.io/daytona/install.sh | sudo bash) && daytona server -y && daytona
```

로컬에서 실행하려면 머신에 Docker가 설치되어 실행 중인지 확인하세요. 다음 명령을 실행하여 워크스페이스를 생성합니다:

```bash
daytona create https://github.com/daytonaio/sample-react-coinbase-connect
```

![Terminal](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/q2e9nwy9dcvjxa36qi58.png)

애플리케이션 시작:

```bash
npm install
npm run dev

# 프로덕션용 빌드
npm run build
npm run preview
```
