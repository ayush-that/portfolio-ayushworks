10x開発者として、直面する最大の生産性キラーは開発環境のセットアップの手間です。予告なしに問題が発生し、フラストレーションにつながるプロセスです。

この問題の解決策を探していたある日、ニューデリーでのDevelopers Club TourミートアップでDaytonaに出会いました。興奮して、[Quira](https://quira.sh/quests/creator/details?questId=23)で彼らのCreator Questを見つけました。

**023: Take Daytona for a spin**チャレンジに挑戦することを決め、このブログでDaytonaでの体験を共有します。

## Daytonaとは？

[Daytona](https://www.daytona.io/)は、セルフホスト型で安全な**オープンソース**の開発環境マネージャーです。

![Daytona GitHub](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/6x6q69ztvir2vwo7yyux.png)

シンプルで手間のかからない**CLIツール**で、ライブラリのダウンロード、ビルド、更新、コンパイル、テストなどの追加ステップを開発プロセスから排除し、**単一のコマンド**でコーディングを開始できます。

## Daytonaを学ぶ

Daytonaの基本的なアイデアを得るために、まず[GitHub](https://github.com/daytonaio/daytona)リポジトリを開き、`README.md`を読みました。活発な開発と13.5kのスターに感銘を受けました。その後、[ドキュメント](https://www.daytona.io/docs/)と[ブログ](https://www.daytona.io/dotfiles)を確認しました。

次に、Daytonaのテーマから提供されているものの中からクエスト用のテーマを選びました。

## アプリの構築

クエストのために、Coinbase Wallet SDKを使用してReactアプリでCoinbase Walletに接続するためのテンプレートを構築しました。

![Coinbase Connect](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/shm1uaimxgzjj6dwwejo.png)

React + Viteプロジェクトをセットアップし、TypeScriptを活用し、Coinbase SDKのドキュメントを速読しました。一晩でMVPが準備できました。shadcn/uiとTailwindCSSでスタイリングし、Vercelにデプロイしました。

Web3プロジェクトのCoinbase接続に焦点を当てていますが、これはshadcn/uiのボイラープレートとしても使用できます（笑）。

## DaytonaでReactとCoinbase SDKを使用する

これはブログで最も包括的なセクションであるべきですが、Daytonaの驚くほどシンプルな性質のおかげで、あまりやることがありません。

まず、Daytonaのインストールは簡単で、Windowsでも同様です。単一のコマンドでインストールが完了します。

```powershell
$architecture = if ($env:PROCESSOR_ARCHITECTURE -eq "AMD64") { "amd64" } else { "arm64" }
md -Force "$Env:APPDATA\bin\daytona"; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.SecurityProtocolType]'Tls,Tls11,Tls12';
Invoke-WebRequest -URI "https://download.daytona.io/daytona/v0.50/daytona-windows-$architecture.exe" -OutFile "$Env:APPDATA\bin\daytona\daytona.exe";
$env:Path += ";" + $Env:APPDATA + "\bin\daytona"; [Environment]::SetEnvironmentVariable("Path", $env:Path, [System.EnvironmentVariableTarget]::User);
daytona serve;
```

LinuxまたはMacOSを使用している場合は、このコマンドでDaytonaをインストールできます：

```bash
(curl -sf -L https://download.daytona.io/daytona/install.sh | sudo bash) && daytona server -y && daytona
```

ローカルで実行するには、マシンにDockerがインストールされ実行されていることを確認してください。次のコマンドを実行してワークスペースを作成します：

```bash
daytona create https://github.com/daytonaio/sample-react-coinbase-connect
```

![Terminal](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/q2e9nwy9dcvjxa36qi58.png)

アプリケーションを起動：

```bash
npm install
npm run dev

# 本番用ビルド
npm run build
npm run preview
```
