Extraire des données de profils LinkedIn peut être incroyablement utile pour divers objectifs, de la recherche à la création de CV. Dans cet article, je vous guiderai à travers le processus d'utilisation de Python avec l'API Proxycurl pour extraire des données d'un profil LinkedIn.

### Étape 1 : Configurer votre Compte Proxycurl

Avant de plonger dans le code, vous devrez créer un compte sur le [site web Proxycurl](https://nubela.co/proxycurl/). Après inscription, vous recevrez une clé API, essentielle pour accéder à l'API Proxycurl.

### Étape 2 : Stocker votre Clé API en Toute Sécurité

Pour garder votre clé API sécurisée, c'est une bonne pratique de la stocker dans un fichier `.env`.

```plaintext
API_KEY=votre_cle_api_ici
```

### Étape 3 : Écrire le Script Python

```python
import requests
import json
from dotenv import load_dotenv
import os
```

- **requests** : Cette bibliothèque vous permet d'envoyer des requêtes HTTP en Python.
- **json** : Utilisé pour gérer les données JSON.
- **dotenv** : Aide à charger les variables d'environnement depuis un fichier `.env`.
- **os** : Fournit des fonctions pour interagir avec le système d'exploitation.

### Étape 4 : Configurer l'URL du Profil LinkedIn et l'Endpoint API

```python
linkedin_profile_url = "https://www.linkedin.com/in/ayush-that/"
api_endpoint = "https://nubela.co/proxycurl/api/v2/linkedin"
```

### Étape 5 : Configurer la Requête API

```python
params = {
    "url": linkedin_profile_url,
    "fallback_to_cache": "on-error",
    "use_cache": "if-present",
    "skills": "include",
    "inferred_salary": "include",
    "extra": "include",
    "personal_email": "include",
}
headers = {
    "Authorization": f"Bearer {API_KEY}",
}
```

### Étape 6 : Envoyer la Requête et Gérer la Réponse

```python
response = requests.get(api_endpoint, params=params, headers=headers)

if response.status_code == 200:
    profile_data = response.json()
    with open("profile_data.json", "w") as json_file:
        json.dump(profile_data, json_file, indent=4)
else:
    print(f"Erreur: {response.status_code}")
```

### Étape 7 : Exécuter le Script

```bash
python app.py
```
