## Was du brauchst

- [Python](https://www.python.org/downloads/) auf deinem Computer installiert
- Eine Webcam
- Einige Bildressourcen für das Spiel vom [GitHub](https://github.com/ayush-that/SoloPong) Repository

## Den Code schreiben

1. **Bibliotheken importieren und Kamera einrichten**

```python
import numpy as np
import cv2 as cv
import cvzone
from cvzone.HandTrackingModule import HandDetector

cap = cv.VideoCapture(0)
cap.set(3, 1280)
cap.set(4, 720)
```

2. **Bildressourcen laden**

```python
imgBg = cv.imread("assets/bg.png")
imgGO = cv.imread("assets/gameover.png")
imgBall = cv.imread("assets/ball.png", cv.IMREAD_UNCHANGED)
imgBat1 = cv.imread("assets/bat1.png", cv.IMREAD_UNCHANGED)
imgBat2 = cv.imread("assets/bat2.png", cv.IMREAD_UNCHANGED)
```

3. **Handdetektor einrichten**

```python
detector = HandDetector(detectionCon=0.8, maxHands=2)
```

4. **Spielvariablen initialisieren**

```python
ballPos = [100, 100]
speedX = 25
speedY = 25
gameOver = False
score = [0, 0]
```
