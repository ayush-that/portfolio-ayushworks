10x डेवलपर्स के रूप में, सबसे बड़ी प्रोडक्टिविटी किलर जिसका हम सामना करते हैं वह है अपना डेवलपर एनवायरनमेंट सेट करने की परेशानी। यह एक ऐसी प्रक्रिया है जहां चीजें बिना किसी नोटिस के गलत हो सकती हैं जो निराशा का कारण बनती है।

मैं इस समस्या का समाधान खोज रहा था जब एक दिन मुझे नई दिल्ली में उनके Developers Club Tour मीटअप में Daytona के बारे में पता चला। उत्साहित होकर, मैंने [Quira](https://quira.sh/quests/creator/details?questId=23) पर उनका Creator Quest देखा।

मैंने **023: Take Daytona for a spin** चैलेंज का प्रयास करने का फैसला किया और इस ब्लॉग में, मैं Daytona के साथ अपना अनुभव साझा करूंगा।

## Daytona क्या है?

[Daytona](https://www.daytona.io/) एक सेल्फ-होस्टेड और सुरक्षित **ओपन-सोर्स** डेवलपमेंट एनवायरनमेंट मैनेजर है।

![Daytona GitHub](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/6x6q69ztvir2vwo7yyux.png)

यह एक सिंपल, बिना परेशानी वाला **CLI टूल** है जो डेवलपमेंट प्रोसेस से लाइब्रेरी डाउनलोड करने, बिल्डिंग, अपडेशन, कंपाइलिंग और टेस्टिंग जैसे अतिरिक्त कदमों को हटाता है और आपको **एक ही कमांड** से कोडिंग शुरू करने देता है।

## Daytona सीखना

Daytona के बारे में बुनियादी जानकारी प्राप्त करने के लिए, मैंने पहले उनकी [GitHub](https://github.com/daytonaio/daytona) रिपॉजिटरी खोली और `README.md` पढ़ी। मैं एक्टिव डेवलपमेंट और 13.5k स्टार्स से प्रभावित था। उसके बाद, मैंने उनके [डॉक्यूमेंटेशन](https://www.daytona.io/docs/) और [ब्लॉग](https://www.daytona.io/dotfiles) को देखा।

फिर मैंने Daytona themes द्वारा प्रदान किए गए विषयों में से क्वेस्ट के लिए बिल्ड करने के लिए एक थीम चुनी।

## ऐप बनाना

क्वेस्ट के लिए, मैंने Coinbase Wallet SDK का उपयोग करते हुए React ऐप के साथ Coinbase Wallet से कनेक्ट करने के लिए एक टेम्पलेट बनाया।

![Coinbase Connect](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/shm1uaimxgzjj6dwwejo.png)

मैंने एक React + Vite प्रोजेक्ट सेट किया, TypeScript का उपयोग किया, Coinbase SDK डॉक्यूमेंटेशन को जल्दी से पढ़ा। एक रात में, मेरे पास एक MVP तैयार था। मैंने इसे shadcn/ui और TailwindCSS के साथ स्टाइल किया और Vercel पर डिप्लॉय किया।

हालांकि यह Coinbase कनेक्ट करने वाले Web3 प्रोजेक्ट्स पर केंद्रित है, इसे shadcn/ui बॉयलरप्लेट के रूप में भी इस्तेमाल किया जा सकता है।

## कम्युनिटी में शामिल होना

Quira प्लेटफॉर्म पर मेरा प्रोजेक्ट (Coinbase Connect) सबमिट होने के बाद मुझे एक स्लैक इनविटेशन और आगे बढ़ने के निर्देशों के साथ एक मेल मिला।

![Mail](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/50s68gh2wcz88zivdoct.png)

मैं रोमांचित था और उनकी Slack कम्युनिटी में शामिल हो गया। मैंने अपना और अपने प्रोजेक्ट का परिचय दिया और आगे के निर्देशों का इंतजार किया।

![Slack](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/fsraml8eor9xj2n6xm2c.png)

एक दिन बाद, Ivan ने मेरे लिए Daytona ऑर्गनाइज़ेशन पर एक खाली [रिपॉजिटरी](https://github.com/daytonaio/sample-react-coinbase-connect) बनाई। मैंने अपना कोडबेस वहां माइग्रेट किया और एक PR raise किया।

अब यहां पहली बाधा, मैं अपने कमिट को `--signoff` करना भूल गया इसलिए इसमें एक एरर था। कुछ रिविज़न के साथ, [मेरा पहला PR](https://github.com/daytonaio/sample-react-coinbase-connect/pull/1) स्वीकार कर लिया गया। अगले दो दिनों में, मैंने [अपना दूसरा PR](https://github.com/daytonaio/sample-react-coinbase-connect/pull/2) किया जहां मैंने सभी हार्डकोडेड कॉन्स्टेंट्स हटाए और [अपना तीसरा PR](https://github.com/daytonaio/sample-react-coinbase-connect/pull/3) कुछ `README.md` परिवर्तनों के साथ।

कुल मिलाकर कम्युनिटी के साथ अनुभव बढ़िया रहा जिसमें Ivan और Jafa ने जहां भी आवश्यक था मुझे गाइड किया।

## Daytona के साथ Coinbase SDK के साथ React का उपयोग करना

यह ब्लॉग का सबसे व्यापक खंड होना चाहिए लेकिन Daytona की अत्यधिक सरल प्रकृति के कारण, आपको ज्यादा कुछ करने की जरूरत नहीं है।

सबसे पहले, Daytona को इंस्टॉल करना बहुत आसान है, Windows पर भी। यह सिंगल कमांड इंस्टॉलेशन है और आप तैयार हैं।

```powershell
$architecture = if ($env:PROCESSOR_ARCHITECTURE -eq "AMD64") { "amd64" } else { "arm64" }
md -Force "$Env:APPDATA\bin\daytona"; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.SecurityProtocolType]'Tls,Tls11,Tls12';
Invoke-WebRequest -URI "https://download.daytona.io/daytona/v0.50/daytona-windows-$architecture.exe" -OutFile "$Env:APPDATA\bin\daytona\daytona.exe";
$env:Path += ";" + $Env:APPDATA + "\bin\daytona"; [Environment]::SetEnvironmentVariable("Path", $env:Path, [System.EnvironmentVariableTarget]::User);
daytona serve;
```

यदि आप Linux या MacOS पर हैं, तो आप इस कमांड से Daytona इंस्टॉल कर सकते हैं:

```bash
(curl -sf -L https://download.daytona.io/daytona/install.sh | sudo bash) && daytona server -y && daytona
```

इसे लोकली चलाने के लिए, सुनिश्चित करें कि आपके मशीन पर Docker इंस्टॉल और रनिंग है। फिर अपना वर्कस्पेस बनाने के लिए निम्न कमांड चलाएं:

```bash
daytona create https://github.com/daytonaio/sample-react-coinbase-connect
```
