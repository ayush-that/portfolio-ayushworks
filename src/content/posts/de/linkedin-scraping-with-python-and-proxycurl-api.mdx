Das Scraping von Daten aus LinkedIn-Profilen kann für verschiedene Zwecke unglaublich nützlich sein, von der Recherche bis zur Erstellung von Lebensläufen.

### Schritt 1: Proxycurl-Konto einrichten

Erstelle ein Konto auf der [Proxycurl-Website](https://nubela.co/proxycurl/).

### Schritt 2: API-Schlüssel sicher speichern

```plaintext
API_KEY=dein_api_schluessel_hier
```

### Schritt 3: Python-Skript schreiben

```python
import requests
import json
from dotenv import load_dotenv
import os
```

### Schritt 4: LinkedIn-Profil-URL und API-Endpunkt einrichten

```python
linkedin_profile_url = "https://www.linkedin.com/in/ayush-that/"
api_endpoint = "https://nubela.co/proxycurl/api/v2/linkedin"
```

### Schritt 5: API-Anfrage konfigurieren

```python
params = {
    "url": linkedin_profile_url,
    "skills": "include",
    "inferred_salary": "include",
}
headers = {
    "Authorization": f"Bearer {API_KEY}",
}
```

### Schritt 6: Anfrage senden und Antwort verarbeiten

```python
response = requests.get(api_endpoint, params=params, headers=headers)

if response.status_code == 200:
    profile_data = response.json()
    with open("profile_data.json", "w") as json_file:
        json.dump(profile_data, json_file, indent=4)
```

### Schritt 7: Skript ausführen

```bash
python app.py
```

### Fazit

Du hast erfolgreich Daten aus einem LinkedIn-Profil mit Python und der Proxycurl API gescraped. Den vollständigen Code findest du auf [GitHub](https://github.com/ayush-that/ResumeGenie).
