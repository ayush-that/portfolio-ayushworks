近年来，科技界一直在热议[人工智能](https://en.wikipedia.org/wiki/Artificial_intelligence)（AI）。AI正在革新各个行业，就像互联网改变了我们的生活一样。在众多可用的AI工具中，[GPT4ALL](https://gpt4all.io/index.html)作为一款免费、注重隐私的聊天机器人脱颖而出，你可以在本地机器上运行它，无需GPU或互联网连接。

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1719211831012/ea4fbdfe-f4ef-4698-8672-c87a4a16dabe.png)

#### 什么是GPT4ALL？

GPT4ALL是**一个在本地运行的AI聊天机器人，不需要互联网连接或GPU。**它设计得用户友好且注重隐私，对于那些有兴趣使用AI但不需要强大硬件的人来说是一个很好的选择。

#### 安装和设置

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1719212632499/da06fcbc-5a4e-4e6a-b0a4-8257af970d17.png)

GPT4ALL适用于Windows、macOS和Linux。以下是如何在不同系统上安装它：

**1\. Windows和macOS：**

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1719212485083/dd9fa038-e177-402b-9620-583e4095a5d1.png)

- 访问[GPT4ALL网站](https://gpt4all.io)。
- 下载适用于你操作系统的安装程序。
- 运行安装程序并按照屏幕上的说明操作。

**2\. Linux：**

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1719212455476/67b1f9fc-5b63-4905-98af-f150ca6ff860.png)

- 对于Ubuntu和基于Debian的发行版，从GPT4ALL网站下载`.run`文件。
- 打开终端并导航到你下载文件的目录。
- 运行以下命令来安装它：

  ```bash
  curl https://gpt4all.io/installers/gpt4all-installer-linux.run --output gpt4all-installer-linux.run

  # 安装依赖项（在Ubuntu/WSL上）
  sudo apt-get install -y libxcb-glx0 libxcb-icccm4 libxcb-image0 libxcb-keysyms1 libxcb-randr0 libxcb-render-util0 libxcb-xkb1 libxcb-shape0 libxkbcommon-x11-0 libxcb-sync1 libxcb-xfixes0-dev libgl-dev

  # 使.run文件可执行
  sudo chmod +x gpt4all-installer-linux.run
  ./gpt4all-installer-linux.run
  ```

- 对于Arch Linux，你可以在AUR（Arch用户仓库）中找到GPT4ALL。使用`yay`等[AUR助手](https://wiki.archlinux.org/title/AUR_helpers)来安装它：
  ```bash
  yay -S gpt4all
  ```
- 或者，你可以通过[Flatpak](https://flatpak.org/setup/Ubuntu)从[Flathub](https://flathub.org/apps/io.gpt4all.gpt4all)安装它：
  ```bash
  flatpak install flathub com.gpt4all.GPT4ALL
  ```

#### 开始使用GPT4ALL

安装后，以下是如何开始使用GPT4ALL：

1. **启动应用程序：**
   - 从你的应用程序菜单打开GPT4ALL。

2. **下载模型：**

   ![](https://cdn.hashnode.com/res/hashnode/image/upload/v1719211416541/863fa1d8-2683-4d88-b927-84a04cea8520.png)
   - 第一次启动GPT4ALL时，你需要下载一个AI模型。这是必需的，因为GPT4ALL在本地运行，不连接互联网进行处理。
   - 选择一个模型，如Meta的"LLaMA"，然后点击下载按钮。这可能需要几分钟。

3. **开始聊天：**
   - 一旦模型下载完毕，你就可以开始新的聊天。
   - 你可以通过标记来保存你的聊天，这有助于跟踪不同的对话。

#### 使用GPT4ALL

GPT4ALL提供各种功能：

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1719211336673/c98b6c17-26ff-42eb-8d43-f7897d71dee5.png)

- **提问：** 例如，你可以让它按字母顺序列出所有哺乳动物。请记住，大型输出可能需要一些时间来生成。
- **编程辅助：** 询问代码示例，如Bash中的for循环或Rust中的递归。GPT4ALL不仅提供代码，还解释它。
- **简单解释：** 请求用简单的话解释，如"像我是5岁一样解释AI"。GPT4ALL可以将复杂的主题分解成易于理解的语言。
- **创意任务：** 你甚至可以让它写诗，比如关于AI的俳句。

#### 自定义GPT4ALL

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1719211455963/8dd5a7a9-9d47-416d-9cc7-ddf3d71e1175.png)

你可以调整设置以改善你的体验：

- **字体大小和主题：** 根据你的偏好更改字体大小并在深色和浅色主题之间切换。
- **CPU线程：** 调整GPT4ALL使用的CPU线程数。这可以提高性能，特别是在多核处理器上。
