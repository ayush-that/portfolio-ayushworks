क्या आपने कभी पिंग पोंग खेलना चाहा है लेकिन कोई साथी नहीं था? खैर, थोड़ी कोडिंग और [कंप्यूटर विज़न](https://en.wikipedia.org/wiki/Computer_vision) के साथ, आप एक सोलो [पिंग-पोंग](https://en.wikipedia.org/wiki/Table_tennis) गेम बना सकते हैं जो आपको खुद के खिलाफ खेलने देता है!

## कोड लिखना

1. **लाइब्रेरीज़ इम्पोर्ट करना और कैमरा सेट करना**

```python
import numpy as np
import cv2 as cv
import cvzone
from cvzone.HandTrackingModule import HandDetector

cap = cv.VideoCapture(0)
cap.set(3, 1280)
cap.set(4, 720)
```

हम आवश्यक लाइब्रेरीज़ इम्पोर्ट करके शुरू करते हैं। हम इमेज प्रोसेसिंग के लिए OpenCV (`cv`), हैंड डिटेक्शन के लिए `cvzone`, और न्यूमेरिकल ऑपरेशंस के लिए [NumPy](https://numpy.org/doc/stable/) (`np`) का उपयोग करते हैं। फिर हम 1280x720 रेजोल्यूशन के साथ वेबकैम सेट करते हैं।

2. **इमेज एसेट्स लोड करना**

```python
imgBg = cv.imread("assets/bg.png")
imgGO = cv.imread("assets/gameover.png")
imgBall = cv.imread("assets/ball.png", cv.IMREAD_UNCHANGED)
imgBat1 = cv.imread("assets/bat1.png", cv.IMREAD_UNCHANGED)
imgBat2 = cv.imread("assets/bat2.png", cv.IMREAD_UNCHANGED)
```

हम बैकग्राउंड, गेम ओवर स्क्रीन, बॉल और बैट्स के लिए इमेजेज़ लोड करते हैं। सुनिश्चित करें कि आपके पास ये इमेजेज़ assets फ़ोल्डर में हैं।

3. **हैंड डिटेक्टर सेट करना**

```python
detector = HandDetector(detectionCon=0.8, maxHands=2)
```

हम 0.8 की डिटेक्शन कॉन्फिडेंस के साथ हैंड डिटेक्टर को इनिशियलाइज़ करते हैं और इसे 2 हाथों तक डिटेक्ट करने की अनुमति देते हैं।

4. **गेम वेरिएबल्स इनिशियलाइज़ करना**

```python
ballPos = [100, 100]
speedX = 25
speedY = 25
gameOver = False
score = [0, 0]
```

हम बॉल की प्रारंभिक स्थिति और उसकी स्पीड सेट करते हैं। हम गेम ओवर फ्लैग और स्कोर भी इनिशियलाइज़ करते हैं।

5. **मुख्य गेम लूप**

```python
while True:
    _, img = cap.read()
    img = cv.flip(img, 1)
    imgRaw = img.copy()
```

हम मुख्य गेम लूप में प्रवेश करते हैं, वेबकैम से फ्रेम पढ़ते हैं, और मिरर इफेक्ट के लिए इसे क्षैतिज रूप से फ्लिप करते हैं। हम मूल फ्रेम की एक कॉपी भी बनाते हैं।

6. **हैंड डिटेक्शन और बैट्स ओवरले करना**

```python
hands, img = detector.findHands(img, flipType=False)
img = cv.addWeighted(img, 0.1, imgBg, 0.9, 0)

if hands:
    for hand in hands:
        x, y, w, h = hand["bbox"]
        h1, w1, _ = imgBat1.shape
        y1 = y - h1 // 2
        y1 = np.clip(y1, 20, 415)

        if hand["type"] == "Left":
            img = cvzone.overlayPNG(img, imgBat1, (59, y1))
            if 59 < ballPos[0] < 59 + w1 and y1 < ballPos[1] < y1 + h1:
                speedX = -speedX
                ballPos[0] += 30
                score[0] += 1

        if hand["type"] == "Right":
            img = cvzone.overlayPNG(img, imgBat2, (1195, y1))
            if 1195 - 50 < ballPos[0] < 1195 and y1 < ballPos[1] < y1 + h1:
                speedX = -speedX
                ballPos[0] -= 30
                score[1] += 1
```

हम फ्रेम में हाथों को डिटेक्ट करते हैं और हाथों की स्थिति के आधार पर बैट्स को ओवरले करते हैं। यदि बॉल बैट से टकराती है, तो इसकी दिशा बदल जाती है और स्कोर अपडेट हो जाता है।

7. **बॉल मूवमेंट और कोलिज़न डिटेक्शन**

```python
if ballPos[0] < 40 or ballPos[0] > 1200:
    gameOver = True
if gameOver:
    img = imgGO
    cv.putText(
        img,
        str(score[1] + score[0]).zfill(2),
        (585, 360),
        cv.FONT_HERSHEY_COMPLEX,
        2.5,
        (200, 0, 200),
        5,
    )

else:
    if ballPos[1] >= 500 or ballPos[1] <= 10:
        speedY = -speedY

    ballPos[0] += speedX
    ballPos[1] += speedY

    img = cvzone.overlayPNG(img, imgBall, ballPos)
    cv.putText(
        img,
        str(score[0]),
        (300, 650),
        cv.FONT_HERSHEY_COMPLEX,
        3,
        (255, 255, 255),
        5,
    )
    cv.putText(
        img,
        str(score[1]),
        (900, 650),
        cv.FONT_HERSHEY_COMPLEX,
        3,
        (255, 255, 255),
        5,
    )
```

हम बॉल की स्थिति को अपडेट करते हैं और स्क्रीन किनारों से टकराव की जांच करते हैं। यदि बॉल सीमा से बाहर जाती है, तो गेम खत्म हो जाता है। अन्यथा, हम बॉल की स्थिति को अपडेट करना जारी रखते हैं और इसे फ्रेम पर ओवरले करते हैं।

img[580:700, 20:233] = cv.resize(imgRaw, (213, 120))

हम गेम फ्रेम प्रदर्शित करते हैं और गेम को रीसेट करने के लिए "R" की प्रेस को हैंडल करते हैं।
