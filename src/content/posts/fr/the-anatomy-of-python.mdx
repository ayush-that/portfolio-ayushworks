Python était mon premier langage de programmation. C'est un langage orienté objet populaire connu pour sa simplicité et sa polyvalence. En tant que langage interprété, Python exécute le code ligne par ligne en utilisant un interpréteur, plutôt que de compiler tout le code en langage machine d'un seul coup.

## Termes de Base à Connaître

### **Éditeur de Code**

C'est l'environnement où le code source est écrit. C'est le point de départ du programme Python.

### **Code Source**

Les instructions lisibles par l'homme écrites dans l'éditeur de code et sauvegardées sous forme de fichier `.py`.

### **Étape de Compilation**

À cette étape, le compilateur Python traduit le code source en bytecode, vérifiant les erreurs de syntaxe en cours de route.

### **Machine Virtuelle Python (PVM)**

La PVM est le moteur d'exécution central de Python. Elle interprète le bytecode et le traduit en code machine que le CPU peut exécuter.

### **Programme en Cours d'Exécution**

Une fois le code machine généré, le CPU l'exécute.
