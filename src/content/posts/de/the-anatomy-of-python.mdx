Python war meine erste Programmiersprache. Es ist eine beliebte objektorientierte Sprache, bekannt für ihre Einfachheit und Vielseitigkeit. Als interpretierte Sprache führt Python Code Zeile für Zeile mit einem Interpreter aus.

## Grundlegende Begriffe

### **Code-Editor**

Dies ist die Umgebung, in der der Quellcode geschrieben wird.

### **Quellcode**

Die für Menschen lesbaren Anweisungen, die im Code-Editor geschrieben und als `.py`-Datei gespeichert werden.

### **Kompilierungsphase**

Der Python-Compiler übersetzt den Quellcode in Bytecode und prüft dabei auf Syntaxfehler.

### **Python Virtual Machine (PVM)**

Die PVM ist die zentrale Laufzeit-Engine von Python. Sie interpretiert den Bytecode und übersetzt ihn in Maschinencode.

### **Laufendes Programm**

Sobald der Maschinencode generiert ist, führt die CPU ihn aus.
