作为10倍开发者，我们面临的最大生产力杀手是设置开发环境的麻烦。这是一个随时可能出错的过程，会导致沮丧。

我一直在寻找解决这个问题的方案，直到有一天我在新德里的Developers Club Tour聚会上遇到了Daytona。兴奋之余，我在[Quira](https://quira.sh/quests/creator/details?questId=23)上看到了他们的Creator Quest。

我决定尝试**023：体验Daytona**挑战，在这篇博客中，我将分享我使用Daytona的经验。

## 什么是Daytona？

[Daytona](https://www.daytona.io/)是一个自托管且安全的**开源**开发环境管理器。

![Daytona GitHub](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/6x6q69ztvir2vwo7yyux.png)

它是一个简单、无忧的**CLI工具**，从开发过程中省去了下载库、构建、更新、编译和测试等额外步骤，让你只需**一条命令**即可开始编码。

## 学习Daytona

为了对Daytona有一个基本的了解，我首先打开他们的[GitHub](https://github.com/daytonaio/daytona)仓库并阅读了`README.md`。我对活跃的开发和13.5k星印象深刻。之后，我浏览了他们的[文档](https://www.daytona.io/docs/)和[博客](https://www.daytona.io/dotfiles)。

然后我从Daytona提供的主题中选择了一个来构建quest项目。

## 构建应用

对于这个quest，我构建了一个用于将React应用连接到Coinbase钱包的模板，使用Coinbase Wallet SDK。

![Coinbase Connect](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/shm1uaimxgzjj6dwwejo.png)

我设置了一个React + Vite项目，使用TypeScript，快速阅读了Coinbase SDK文档。在一个晚上内，我就准备好了一个MVP。我用shadcn/ui和TailwindCSS进行样式设计，并部署到Vercel。

虽然它专注于涉及连接Coinbase的Web3项目，但它也可以用作shadcn/ui样板，哈哈。

![邮件](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/50s68gh2wcz88zivdoct.png)

## 使用Daytona运行React与Coinbase SDK

这应该是博客中最全面的部分，但由于Daytona极其简单的特性，你不需要做太多事情。

首先，安装Daytona非常简单，即使在Windows上也是如此。这是一条命令安装，然后你就完成了。

```powershell
$architecture = if ($env:PROCESSOR_ARCHITECTURE -eq "AMD64") { "amd64" } else { "arm64" }
md -Force "$Env:APPDATA\bin\daytona"; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.SecurityProtocolType]'Tls,Tls11,Tls12';
Invoke-WebRequest -URI "https://download.daytona.io/daytona/v0.50/daytona-windows-$architecture.exe" -OutFile "$Env:APPDATA\bin\daytona\daytona.exe";
$env:Path += ";" + $Env:APPDATA + "\bin\daytona"; [Environment]::SetEnvironmentVariable("Path", $env:Path, [System.EnvironmentVariableTarget]::User);
daytona serve;
```

如果你使用的是Linux或MacOS，你可以用这个命令安装Daytona：

```bash
(curl -sf -L https://download.daytona.io/daytona/install.sh | sudo bash) && daytona server -y && daytona
```

要在本地运行，请确保你的机器上已安装并运行Docker。然后运行以下命令来创建你的工作区：

```bash
daytona create https://github.com/daytonaio/sample-react-coinbase-connect
```

![终端](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/q2e9nwy9dcvjxa36qi58.png)

启动应用程序：

```bash
npm install
npm run dev

# 构建生产版本
npm run build
npm run preview
```
