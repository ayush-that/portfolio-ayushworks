Python मेरी पहली प्रोग्रामिंग लैंग्वेज थी। यह एक लोकप्रिय ऑब्जेक्ट-ओरिएंटेड लैंग्वेज है जो अपनी सरलता और बहुमुखीता के लिए जानी जाती है। एक इंटरप्रेटेड लैंग्वेज के रूप में, Python एक इंटरप्रेटर का उपयोग करके कोड को लाइन दर लाइन निष्पादित करता है, न कि पूरे कोड को एक बार में मशीन लैंग्वेज में कंपाइल करता है।

इस आर्टिकल में, हम स्रोत कोड लिखने से लेकर इसके निष्पादन तक Python की आंतरिक कार्यप्रणाली को एक्सप्लोर करेंगे। इन चरणों को समझना डेवलपर्स को एक प्रोग्रामिंग लैंग्वेज के रूप में Python की दक्षता और शक्ति की सराहना करने में मदद करता है।

## जानने के लिए बुनियादी शब्द

Python की कार्यप्रणाली में गहराई से जाने से पहले, आइए पहले कुछ बुनियादी शब्दों को समझते हैं:

### **कोड एडिटर**

यह वह वातावरण है जहां स्रोत कोड लिखा जाता है। यह Python प्रोग्राम का शुरुआती बिंदु है, जहां डेवलपर्स मानव-पठनीय प्रारूप में निर्देश लिखते हैं।

### **स्रोत कोड**

कोड एडिटर में लिखे गए मानव-पठनीय निर्देश और `.py` फाइल के रूप में सेव किए गए। यह फाइल वह कोड होता है जिसे Python कंपाइलर प्रोसेस करेगा।

### **कंपाइलेशन स्टेज**

इस स्टेज में, Python कंपाइलर स्रोत कोड को बाइट कोड में अनुवाद करता है, इस प्रक्रिया में सिंटैक्स त्रुटियों की जांच करता है। यदि कोई त्रुटि नहीं मिलती है, तो बाइट कोड युक्त एक `.pyc` फाइल जेनरेट की जाती है।

### **Python Virtual Machine (PVM)**

PVM Python का कोर रनटाइम इंजन है। यह बाइट कोड को इंटरप्रेट करता है और इसे मशीन कोड में अनुवाद करता है जिसे CPU निष्पादित कर सकता है। PVM सुनिश्चित करता है कि Python कोड को Python इंटरप्रेटर वाली किसी भी मशीन पर निष्पादित किया जा सकता है, जिससे Python अत्यधिक पोर्टेबल बन जाता है।

### **चलने वाला प्रोग्राम**

एक बार मशीन कोड जेनरेट हो जाने के बाद, CPU इसे निष्पादित करता है, और प्रोग्राम मूल Python स्क्रिप्ट द्वारा निर्दिष्ट कार्य और कंप्यूटेशन्स को परफॉर्म करता है।

## Python की निष्पादन प्रक्रिया

Python कोड कंप्यूटर के CPU द्वारा निष्पादित होने से पहले कई चरणों से गुज़रता है। यहां Python कोड को एक्सीक्यूटेबल कोड में कैसे परिवर्तित किया जाता है इसका स्टेप-बाय-स्टेप विवरण दिया गया है:

![कोड एडिटर](https://cdn.hashnode.com/res/hashnode/image/upload/v1715955234708/e46fbbb2-e325-4197-ae40-7511d6bfebb9.png)

1. ### स्रोत कोड लिखना

   ![](https://live.staticflickr.com/4836/44661017890_eb4f288ffe_b.jpg)

   किसी भी Python प्रोग्राम का पहला कदम कोड एडिटर में स्रोत कोड लिखना है। यह मानव-पठनीय कोड है, Python के सिंटैक्स नियमों के अनुसार लिखा गया। एक बार लिखा जाने के बाद, यह स्रोत कोड `.py` फाइल के रूप में सेव किया जाता है।

2. ### बाइट कोड में कंपाइलेशन

   ![](https://prepinstadotcom.s3.ap-south-1.amazonaws.com/wp-content/uploads/2020/07/Pythonbyte.webp)

   जब .py फाइल निष्पादित होती है, तो Python कंपाइलर पहले स्रोत कोड को पढ़ता है। इस स्टेज के दौरान, कंपाइलर किसी भी सिंटैक्स त्रुटि की जांच करता है। यदि कोई त्रुटि नहीं मिलती है, तो कंपाइलर स्रोत कोड को बाइट कोड में परिवर्तित करता है, जिससे एक .pyc फाइल बनती है। मशीन कोड के विपरीत, बाइट कोड CPU द्वारा सीधे निष्पादित नहीं किया जा सकता है लेकिन यह एक मध्यवर्ती प्रतिनिधित्व के रूप में काम करता है।

3. ### **Python Virtual Machine (PVM) द्वारा इंटरप्रेटेशन**

   बाइट कोड फिर Python Virtual Machine (PVM) को फीड किया जाता है। PVM एक इंटरप्रेटर है जो बाइट कोड को पढ़ता है और इसे मशीन-एक्सीक्यूटेबल कोड में अनुवाद करता है। यह अनुवाद लाइन दर लाइन होता है, जिसका अर्थ है कि PVM बाइट कोड की प्रत्येक लाइन को अनुक्रमिक रूप से इंटरप्रेट और निष्पादित करता है।

4. ### CPU द्वारा निष्पादन

   अंत में, PVM द्वारा तैयार किया गया मशीन कोड CPU द्वारा निष्पादित किया जाता है। CPU, जो केवल बाइनरी कोड को समझता है, मूल Python प्रोग्राम में निर्दिष्ट कार्य और कंप्यूटेशन्स को परफॉर्म करता है, वांछित आउटपुट देता है।
