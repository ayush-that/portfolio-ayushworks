En tant que développeurs 10x, le plus grand tueur de productivité auquel nous faisons face est le tracas de configurer notre environnement de développement. C'est un processus où les choses peuvent mal tourner sans préavis, ce qui mène à la frustration.

Je cherchais une solution à ce problème quand un jour j'ai découvert Daytona lors de leur meetup Developers Club Tour à New Delhi. Excité, j'ai vu leur Creator Quest sur [Quira](https://quira.sh/quests/creator/details?questId=23).

J'ai décidé de tenter le défi **023 : Essayez Daytona** et dans ce blog, je partagerai mon expérience avec Daytona.

## Qu'est-ce que Daytona ?

[Daytona](https://www.daytona.io/) est un gestionnaire d'environnement de développement **open-source**, auto-hébergé et sécurisé.

![Daytona GitHub](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/6x6q69ztvir2vwo7yyux.png)

C'est un **outil CLI** simple, sans tracas, qui élimine les étapes supplémentaires comme le téléchargement de bibliothèques, la construction, la mise à jour, la compilation et les tests du processus de développement et vous permet de commencer à coder avec **une seule commande**.

## Apprendre Daytona

Pour avoir une idée de base sur Daytona, j'ai d'abord ouvert leur dépôt [GitHub](https://github.com/daytonaio/daytona) et lu le `README.md`. J'ai été impressionné par le développement actif et les 13.5k étoiles.

## Construire l'Application

Pour la quête, j'ai construit un template pour connecter une application React avec Coinbase Wallet, en utilisant le SDK Coinbase Wallet.

![Coinbase Connect](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/shm1uaimxgzjj6dwwejo.png)

J'ai configuré un projet React + Vite, utilisé TypeScript, lu rapidement la documentation du SDK Coinbase. En une nuit, j'avais un MVP prêt.

## Rejoindre la Communauté

Après que mon projet (Coinbase Connect) a été soumis sur la plateforme Quira, j'ai reçu un mail avec une invitation Slack et des instructions pour progresser davantage.

## Utiliser React avec Coinbase SDK avec Daytona

Installer Daytona est un jeu d'enfant, même sur Windows. C'est une installation en une seule commande et vous avez terminé.

```bash
(curl -sf -L https://download.daytona.io/daytona/install.sh | sudo bash) && daytona server -y && daytona
```

Pour l'exécuter localement, assurez-vous d'avoir Docker installé et en cours d'exécution sur votre machine :

```bash
daytona create https://github.com/daytonaio/sample-react-coinbase-connect
```

Démarrer l'Application :

```bash
npm install
npm run dev
```
